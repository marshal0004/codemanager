## 🧹 Step 2: Cleaning the Dataset

### Why clean?
- Your dataset might contain broken code, duplicates, or irrelevant text.

### Basic Cleaning Steps:
```python
def clean_code(example):
    content = example["content"]
    # Remove empty code
    if not content or len(content.strip()) < 20:
        return False
    # Optional: Remove large files
    if len(content) > 5000:
        return False
    return True

cleaned_dataset = dataset.filter(clean_code)
```

### Tokenization:
```python
def tokenize(example):
    return tokenizer(example["content"], truncation=True, padding="max_length", max_length=512)

tokenized_dataset = cleaned_dataset.map(tokenize, batched=True)
```

## ✅ Training with QLoRA

```python
from transformers import AutoModelForCausalLM, AutoTokenizer, TrainingArguments, Trainer
from peft import get_peft_model, LoraConfig, TaskType
from datasets import load_dataset

model_id = "bigcode/starcoderbase-1b"
tokenizer = AutoTokenizer.from_pretrained(model_id)
model = AutoModelForCausalLM.from_pretrained(model_id, load_in_4bit=True, device_map="auto")

# Apply LoRA
lora_config = LoraConfig(
    r=8,
    lora_alpha=32,
    target_modules=["q_proj", "v_proj"],
    lora_dropout=0.05,
    bias="none",
    task_type=TaskType.CAUSAL_LM
)

model = get_peft_model(model, lora_config)

# Load and clean dataset
dataset = load_dataset("codeparrot/codeparrot-clean", split="train[:1%]")
dataset = dataset.filter(lambda example: example["content"] is not None)
tokenized = dataset.map(lambda x: tokenizer(x["content"], truncation=True, padding="max_length", max_length=512), batched=True)

# Training Settings
training_args = TrainingArguments(
    output_dir="./code-model",
    per_device_train_batch_size=1,
    gradient_accumulation_steps=8,
    logging_dir="./logs",
    save_steps=100,
    num_train_epochs=2,
    fp16=True,
)

trainer = Trainer(
    model=model,
    args=training_args,
    train_dataset=tokenized,
)

trainer.train()
```

# 🚀 PHASE 6: Deploy Locally (Gradio or FastAPI)

### ✅ Using Gradio:

```bash
pip install gradio
```

```python
import gradio as gr

def code_gen(prompt):
    input_ids = tokenizer(prompt, return_tensors="pt").input_ids.cuda()
    output = model.generate(input_ids, max_new_tokens=200)
    return tokenizer.decode(output[0], skip_special_tokens=True)

gr.Interface(fn=code_gen, inputs="text", outputs="text").launch()
```
